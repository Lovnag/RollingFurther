В данном модуле реализованы векторы и основные операции над ними.

Для создания вектора в пространстве размерности N необходимо иметь массив N элементов, которые будут компонентами вектора. Таким образом, чтобы создать вектор а с помощью массива massive, необходимо исполнить следующее:
Vector a = new Vector(massive);

Данная реализация поддерживает сложение и разность векторов, умножение вектора на число, нахождение модуля вектора, нахождение скалярного и векторного произведений векторов. Все операции проводимы над векторами размерности N.
Компоненты вектора, его модуль и скалярное произведение векторов реализованы как вещественные числа. Сложение/разность векторов, умножение вектора на число и векторное произведение возвращают векторы.
Все методы, параметрами которых являются векторы, проверяют их размерность на равенство с размерностью того вектора, методом которого они были вызваны. При неравенстве возникает ошибка.
 С помощью методов getDimension() и getTheVector() можно узнать размерность вектора и его компоненты.

Метод theModule() возвращает модуль вектора. 

Метод theAddition(Vector vector2) возвращает сумму векторов, при приеме еще одного вектора как параметра.

Метод theSubtraction(Vector vector2) возвращает разность векторов, при приеме еще одного вектора как параметра.

Метод theMultiplication(double modifier) возвращает умноженный на число вектор, при приеме вещественного числа как параметра.

Метод theScalarMultiplication(Vector vector2) возвращает скалярное произведение векторов, при приеме еще одного вектора как параметра.

Метод theVectorMultiplication(Vector[] vectors) возвращает векторное произведение векторов. Стоит отметить, что, так как данная реализация поддерживает вычисление векторного произведения для векторов размерности N, то как параметр необходимо передавать массив из N-2 векторов. Если в количество векторов в массиве не соответствует N-2, возникнет ошибка.
